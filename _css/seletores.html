No CSS nós possuímos os Seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID(#): Busca elementos através do atributo "id"
  • Seletor por Classe(.): Busca por através do atributo "class"
  • Seletor Universal(*):Seleciona todo os elementos do HTML
  • Seletor de Atributo([atrib]):Seleciona elemetos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valos específico
